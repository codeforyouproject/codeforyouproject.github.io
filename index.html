WANT TO LEARN HTML?

Codeforyou is an online computer science education platform that teaches students how to code in HTML/CSS/JavaScript. Our interactive code sketches allow students to learn by doing, and our tutorials provide step-by-step guidance to create their own code sketches using Codepen. Our focus is on making coding accessible and fun for everyone.

HTML

Learn the basics of HTML and start building your own codesketches or web pages. Our activities cover the fundamental concepts of HTML, including tags, attributes, and structure. By the end of the course, you'll have created your own code sketches from scratch.

CSS

Take your code sketches to the next level with CSS styling. Our activities cover the basics of CSS, including selectors, properties, and values, which will help you create beautiful and responsive designs.

JavaScript

Get started with JavaScript and learn how to add interactivity to your code sketches and web pages. Our activities cover the basics of JavaScript, including variables, functions, and events. By the end of the course, you'll be able to create dynamic and engaging web pages.

Applications

Apply your skills to real-world projects with our code sketch activities. Our activities cover topics such as responsive design, web accessibility, and web performance. By the end of the exercises, you'll have built your own fully functional code sketches.

About Me

I’m Liv, and I’m a high school senior from NYC. I first started taking Computer Science in the 9th grade, and I quickly became fascinated with code. I’ve taken CompSci every year since, and even competed in Indiana University’s bias on social media hackathon and datathon in 2023.

I’ve noticed a lack of free, simple practice options for code learners on the internet. Lacking them in the ninth grade, I was able to acquire the skills to manipulate code, but not to build it. In tenth grade I learned Java, but it took me until 11th grade to truly understand HTML/CSS. Even then, I essentially had to teach it to myself because there was still a lack of these resources.

That’s why I’ve built this website. It’s completely free to use. If you’re a student, use it as much as you like. If you’re a teacher, use it for your class! And please always feel free to contact me with any questions, suggestions, or contributions. Happy coding!
