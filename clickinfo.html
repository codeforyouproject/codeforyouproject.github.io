An event listener is a piece of code that listens for an event. An “event” could be a mouse press, move, input, or something else. This piece of code uses a mouse press. When the mouse is pressed, a “div” element is created and populated onto the page. In this case, we’ve added a pink circle, but you can make the image anything you want.
